#### client service installation
To run client service you need node version 22 or later
1. Go to the client folder and install essential packages

**npm**

```
    npm install
```

**yarn**

```
    yarn install
```

**pnp**

```
    pnpm install
```
2. Create the .env file

```
    HOST_URL=http://127.0.0.1:8080
```

3. Now you can run the client in dev mode using
```
    npm run dev
```

4. Or build and deploy it to production servers
